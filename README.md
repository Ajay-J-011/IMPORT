## How does this Game Works
1. You choose either **Rock (0)**, **Paper (1)**, or **Scissors (2)**.
2. The computer randomly picks one of the three choices.
3. The winner is decided based on the classic rules:
   - Rock beats Scissors
   - Paper beats Rock
   - Scissors beats Paper
   - Same choice = Draw
